ให้หาความเหมือนและความแตกต่างระหว่าง 2 tag ที่กำหนดให้

Tag Name Similar Difference
hr & br - - hr เป็นคำสั่งสร้างเส้นตรง
- เป็นคำสั่งเว้นบรรทัดใหม่
ul & ol เป็นคำสั่งสร้างลิสต์เหมือนกัน - ul (Unordered List) ลิสต์จะไม่มีหมายเลขกำกับเป็นข้อๆ
- ol (Ordered List) ลิสต์มีหมายเลขกำกับเป็นข้อๆ
select & checkbox & radio button สร้างคำสั่งเลือก - select ใช้สร้าง drop-down list
- checkbox ใช้สร้างเครื่องหมายติ๊กหัวข้อที่เราเลือก
- radio button ใช้สร้างการเลือกแค่หนึ่งหัวข้อจากหัวข้อทั้งหมด
div & span มีความเกี่ยวข้องกับการใช้คำสั่ง section ของหน้าเว็บ หรือส่วนใดส่วนหนึ่งของเว็บ - div เป็น block-level element ใช้แบ่ง section ในหน้าเว็บ
- span เป็น inline container ใช้ทำหัวข้อเป็นในแนวนอน
script & style เป็นแท็กใช้สร้างครอบคลุมภาษาในการเขียนเว็บเหมือนกัน - script ใช้ครอบคลุมภาษา JavaScript
- style ใช้ครอบคลุมภาษา CSS


ให้หาว่า input type มีอะไรบ้าง แต่ละ type แตกต่างกันอย่างไร

Input Type Difference
button คำสั่งสร้างปุ่ม button
checkbox ใช้สร้างเครื่องหมายติ๊กหัวข้อที่เราเลือก
color กำหนดค่าสีใน input field
date ใช้กำหนดวันที่เกิด ใน input field
datetime-local ใช้กำหนดวันที่เกิด ใน input field โดยไม่มี time zone
email ใช้ใส่ค่า email address ใน input field
file สร้างคำสั่งปุ้ม Browse สำหรับการอัพโหลดไฟล์
month ใช้ให้ user กำหนดค่าเดือนและปีที่เกิด
number ใส่ค่าตัวเลขใน input field
password ใส่ค่า password ใน input field
radio ใช้สร้างปุ่มเลือกหัวข้อจากหัวข้อทั้งหมด
reset ทำให้ค่ากรอกข้อมูล reset ใหม่หมดเป็น default
submit ปุ่มคลิก submit
text ใช้พิมพ์ข้อความใน input field
url ใส่ข้อมูล URL link ใน input field
week ใช้ให้ user เลือกสัปดาห์และปี
